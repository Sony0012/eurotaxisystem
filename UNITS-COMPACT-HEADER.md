- **Font Size**: `text-sm` → `text-xs` (25% reduction)
- **Margin**: Removed `mt-3` for tighter spacing

**4. Summary Stats:**
- **Padding**: `p-6` → `p-4` (33% reduction)
- **Card Padding**: `p-4` → `p-3` (25% reduction)
- **Icon Padding**: `p-2` → `p-1.5` (25% reduction)

- **Number Size**: `text-2xl` → `text-lg` (20% reduction)
- **Gap**: `gap-6` → `gap-4` (33% reduction)

**5. Units Grid Area:**

- **Bottom Padding**: `pb-6` → `pb-4` (33% reduction)
- **Loading Padding**: `py-20` → `py-16` (20% reduction)
